# FAQ

## What is AIChatbot?

AIChatbot is a hosted AI sales platform for businesses that want a public chat experience plus an admin workspace for products, knowledge, leads, prompts, analytics, and channel operations.

## Who is it for?

Primary audiences include:

- ecommerce teams,
- sales and support operators,
- businesses managing customer conversations across multiple channels.

## Is AIChatbot only a website chatbot?

No. It combines:

- a public marketing and chat layer,
- account and onboarding flows,
- an admin and operations layer.

## Does it support multiple channels?

Yes. The product is designed around a multi-channel sales workflow that can include web chat and external messaging platforms depending on the configured deployment.

## Does it include an admin panel?

Yes. The live product includes admin-side surfaces for dashboard, products, knowledge, prompts, rules, leads, channels, analytics, billing, and team management.

## Does this repository contain the source code?

No. This is a public product-information and support repository only.

## Is it self-hosted by end users?

No public self-hosted distribution flow is documented here.
