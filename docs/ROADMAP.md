# Public Roadmap

This is a lightweight public roadmap for AIChatbot.

## Current product shape

- hosted marketing and login surface
- live chat experience
- admin dashboard
- products, knowledge, prompts, rules, leads, channels, analytics, billing, and team pages

## Likely next improvements

- deeper channel integration maturity
- better analytics and conversion visibility
- stronger onboarding and self-service setup flows
- richer admin-side configuration for prompts, skills, and rules
- more polished deployment and customer handoff workflows

## Scope note

This roadmap is directional only. It is not a release guarantee.
